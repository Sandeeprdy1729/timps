Getting Started

TIMPS is an AI coding agent with persistent memory that remembers your preferences, codebase, and workflows across sessions.

Installation

npm install -g timps-code
# or
brew install timps

Configuration

Create a config file at ~/.timps/config.json:

{
  "model": "claude-3-opus",
  "memory": {
    "episodic": true,
    "semantic": true
  }
}

Plugins

# Install plugin
timps plugin install git

# List plugins
timps plugin list

API Reference

Core Methods

  • timps.run(task) - Run a coding task
  • timps.chat(message) - Interactive chat
  • timps.memory.get(key) - Get from memory
  • timps.memory.set(key, value) - Store in memory

Troubleshooting

Common Issues

  • API key not found: Set environment variable
  • Connection timeout: Check network
  • Memory errors: Run timps doctor
