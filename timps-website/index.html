How It Works

Your project.
Remembered forever.

1. You give TIMPS a task

In the CLI, VS Code chat, or via REST API. Natural language — no special syntax required.

2. TIMPS loads your memory

Episodic memory surfaces relevant past sessions. Semantic memory loads your project conventions. Working memory sets the current context.

3. Agent loop runs with tools

25+ tools available: read/write files, run shell commands, git operations, web search, GitHub API, Docker, tests, and more. Self-corrects on failure.

4. Memory updated automatically

What was built, what failed, what patterns were used — all stored. Next session starts smarter than the last.
