TIMPS Documentation

Everything you need to install, configure, and build with the full TIMPS ecosystem.

Installation

Requires Node.js 18+.

npm install -g timps-code timps --version

With Ollama (free, recommended)

ollama pull qwen2.5-coder:7b ollama pull qwen2.5-coder:14b ollama pull sandeeprdy1729/timps-coder
No API key needed with Ollama. TIMPS auto-detects it when Ollama is running.

Quick Start

timps timps "add error handling to src/api.ts" timps --provider claude "refactor this module" timps --provider timps-coder "fix the async bug in queue.ts"

Memory System

Three layers survive every restart:

working
Working Memory

Current goal, active files, recent errors. Lives in RAM. Resets on exit.

episodic
Episodic Memory

Summaries of past sessions: what was built, what failed. Persisted to disk.

semantic
Semantic Memory

Permanent facts: patterns, conventions, architecture decisions.

Memory Schema

~/.timps/memory/<project-hash>/episodes.jsonl ~/.timps/memory/<project-hash>/semantic.json
Do not manually edit these files. Use /memory commands instead. Schema is backwards-compatible across versions.

SkillForge Composer

Evolutionary optimizer for composing multiple skills efficiently.

  • Builds a synergy graph — nodes are skills, edges model compatibility
  • Runs genetic algorithm (selection → crossover → mutation → pruning)
  • Prunes overlapping content — redundancy drops from 38% to 4%
Benchmark: 72% better token efficiency (1.0x → 1.72x). Success rate 74% → 91%. Redundancy 38% → 4%.
python3 scripts/skillforge_benchmark.py

TIMPS Swarm

22-agent system running on Ollama + LangGraph. No cloud APIs. 10 SDLC agents + 12 Mac health agents.

git clone https://github.com/Sandeeprdy1729/timps-swarm cd timps-swarm make all open http://localhost:3000

Run tasks

python3 give_work.py "Write a REST API for auth" python3 give_work.py "Create a job queue" -l python python3 give_work.py --spawn code_generator qa_tester "Write and test" python3 give_work.py "My laptop fan is always running" python3 give_work.py --health

API key auth (team use)

python3 give_work.py --keygen "team-key" python3 give_work.py --keys python3 give_work.py --revoke timps-sk-xxxx
