timps-code CLI

The main TIMPS CLI coding agent — the thing most people install first

⭐ GitHub 📦 npm 📖 Docs

timps-code is a terminal-based AI coding agent with a 3-layer persistent memory system, 25+ built-in tools, multi-provider support, and a self-correcting agent loop. It's the open-source alternative to Claude Code that doesn't forget everything when you close the terminal.

Key features

  • 3-layer memory: working, episodic, semantic
  • 25+ tools: file ops, shell, git, web search, tests, GitHub
  • Multi-provider: Ollama (free), Claude, GPT-4, Gemini, OpenRouter, DeepSeek
  • Self-correcting loop: retries 3x with revised approach on failure
  • Skills system: install domain expertise from SkillGalaxy
  • Memory branching: safe experimentation with /branch
  • One-shot and interactive modes
  • TUI (Ink/React terminal UI) with streaming output

Quick install

$ npm install -g timps-code
$ ollama pull qwen2.5-coder:7b
$ timps
✓ Ollama detected
✓ Memory loaded
Ready. Type your task →

TIMPS-Coder Model

Fine-tuned 0.5B coding model with 20 LoRA adapters for bug patterns

⭐ GitHub 🤗 HuggingFace 🦙 Ollama

TIMPS-Coder is a fine-tuned version of Qwen2.5-Coder-0.5B-Instruct trained with Unsloth on a MacBook M2 Air. Despite its small size, it achieves 92% accuracy on a held-out bug-fix benchmark — beating models 10x its size on specialized tasks.

The secret is 20 domain-specific LoRA adapters, each trained on a specific class of bugs. The right adapter is selected automatically based on the error context.

Model specs

  • Base: Qwen2.5-Coder-0.5B-Instruct
  • Training: Unsloth (4-bit QLoRA)
  • Hardware: MacBook M2 Air
  • Accuracy: 92% on bug-fix benchmark
  • 20 LoRA adapters (one per bug class)
  • Available: Ollama, HuggingFace, MLX

LoRA adapters (20 bug patterns)

  • java_npe — NullPointerException
  • python_asyncio — async/await mistakes
  • js_null_check — null/undefined errors
  • react_hooks — hooks rules violations
  • sql_injection — SQL security
  • xss_vulnerability — XSS prevention
  • memory_leak — resource cleanup
  • race_condition — concurrency bugs
  • + 12 more adapters
# Run via Ollama ollama run sandeeprdy1729/timps-coder timps --provider timps-coder "fix the async bug in src/queue.ts" # Or from HuggingFace from transformers import AutoModelForCausalLM, AutoTokenizer model = AutoModelForCausalLM.from_pretrained("sandeeprdy1729/TIMPS-Coder-0.5B")

SkillGalaxy Marketplace

10,171+ community skills for Claude Projects and any MCP client

TIMPS Swarm Multi-Agent

22 specialized agents covering the full SDLC pipeline and computer health

⭐ GitHub

TIMPS Swarm is a Python-based multi-agent system built on LangGraph and Ollama. Give it a feature request and it handles the entire software development lifecycle automatically — from writing requirements to deploying a Docker container.

The computer health subsystem is a unique 12-agent cluster that diagnoses and fixes macOS problems: disk health, memory pressure, slow startup, network issues, and more.

SDLC Pipeline (10 agents)

  • PM Agent — writes user stories and acceptance criteria
  • Architect Agent — proposes system design and ADRs
  • Coder Agent — writes the implementation
  • Reviewer Agent — code review and refactoring
  • QA Agent — writes and runs tests
  • Security Agent — OWASP audit and pen testing
  • Docs Agent — README, API docs, inline comments
  • DevOps Agent — Dockerfile, CI/CD, deployment
  • Observer Agent — metrics, tracing, logging
  • Coordinator — orchestrates the full pipeline

Computer Health (12 agents)

  • Disk health analyzer
  • Memory pressure monitor
  • Startup items optimizer
  • Network diagnostics
  • DNS troubleshooter
  • Battery health checker
  • Process bloat detector
  • Log analyzer
  • System updates checker
  • Permission repair agent
  • Thermal manager
  • Backup status verifier

timps-mcp MCP Server

Give Claude Code and Cursor the persistent memory they were born without

⭐ GitHub 📦 npm 📖 Setup

timps-mcp is a Model Context Protocol server that adds TIMPS's persistent memory system to any MCP-compatible tool. Install it once and Claude Code, Cursor, Windsurf, and GitHub Copilot all get long-term project memory.

The 20 tools range from basic memory operations (store, recall, list) to advanced intelligence tools like the bug prophet, contradiction detector, and burnout seismograph.

npm install -g timps-mcp # Claude Code (add to claude_desktop_config.json) { "mcpServers": { "timps": { "command": "timps-mcp" } } } # Cursor: Settings → MCP → Add → Command: timps-mcp

sandeep-ai Server

Full REST API server and web dashboard for building custom AI pipelines

⭐ GitHub 📖 Docs

sandeep-ai is the server component of the TIMPS ecosystem. It exposes the agent loop, memory system, and intelligence tools as a REST API, making it easy to build custom AI-powered applications or host TIMPS as a managed service.

What's included

  • Express REST API with 17 intelligence tools
  • Web dashboard at /dashboard
  • Embedding-based long-term memory
  • Multi-provider support (same as CLI)
  • Docker-ready with multi-arch builds
  • Vercel deployment config included

REST API endpoints

  • POST /api/agent/run
  • GET /api/memory
  • POST /api/memory/save
  • GET /api/tools
  • POST /api/skills/install
  • GET /api/episodes
