Get started

Running in under 2 minutes

Requires Node.js 18+ and Docker. That's it.

1
Clone & install
One command installs everything — Postgres, Qdrant, dependencies.
curl -fsSL .../install.sh | bash
2
Add API key
Get a free key at openrouter.ai. Drop it in .env. Any model works.
OPENROUTER_API_KEY=sk-or-v1-...
3
Run & seed
Start the server and seed demo data. Dashboard shows live intelligence instantly.
npm run server
npm run seed
